
		ROM イメージ、DISKイメージの吸いだし方法

						2003/06/18
						福永省三
						e-mail : fukuchan@mh1.117.ne.jp


  現在のところ、QUASI88 の配布物には、ROMデータや、ディスクデータの吸いだし
のツールが入っていません。つまり、QUASI88 を動かしたいひとは、なんとかして
自力でイメージを入手しなくてはなりません。

ここでは、イメージを自力で入手する方法について述べます。

●「PC8801mk-2SR emulator」附属のツールを使用する方法

  この方法は、
  	A ) 5 インチ 2D のディスクの読める NEC PC-98x1 を持っている。
	B ) 5 インチ のディスクの読める NEC PC-98x1 を持っていて、かつ、
	    mシリーズのPC-8801 を持っている。
  のいずれかに該当する場合、可能です。

	以下の Web から、「PC8801mk-2SR emulator」をダウンロードし、
	附属のドキュメントのとおりにイメージを吸いだしてください。

		http://www1.plala.or.jp/aoto/pc88emu.htm



●「M88」附属のツールを使用する方法

  この方法は、
	A ) 5 インチ 2D のディスクの読める PC-AT互換機を持っている。
	B ) Windows機 と 88 が近くにある
  のいずれかに該当する場合、可能です。
  
	以下の Web から、「M88 ツール」をダウンロードし、
	附属のドキュメントのとおりにイメージを吸いだしてください。

		http://retropc.net/cisc/m88/
