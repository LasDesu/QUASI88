
		QUASI88 のコンパイル方法  for Macintosh + MPW

						2007/05/09
						福永省三
						e-mail : fukuchan@mh1.117.ne.jp
-------------------------------------------------------------------------------

●ここでは Macintosh (Classic) 環境の人向けに、QUASI88 を MPW を使って
  コンパイルする方法を説明します。

  ここでは、MacOS9 での例を説明します。
  MacOS X の場合は、 BUILD-UNIX.TXT を参考にしてください。


	予め、MPW をインストールしておいてください。
	MPW は、アップルが配布しているフリーのコンパイル環境で、以下の URL より
	入手可能です。(2006/09/10現在)

	        ftp://ftp.apple.com/developer/Tool_Chest/Core_Mac_OS_Tools/MPW_etc./MPW-GM_Images/MPW-GM.img.bin



	1) QUASI88のソースを、展開してください。
	   ソースファイルの入っている src というフォルダおよび、
	   obj という空っぽのフォルダ、MWP用のプロジェクトファイル 
	   QUASI88.make などが展開されます。

	   QUASI88.make ではなく、 QUASI88.make.sit や、 QUASI88.make.sit.hqx
	   というファイルがある場合は、 StuffIt などで展開しておいてください。


	2) SDLのホームページ http://www.libsdl.org/ より、
	   SDLの開発者向けライブラリ (Development Libraries) の、
	   MacOS (Classic) 版を入手します。

	   2006/09/10現在、SDLのバージョンは 1.2.11 です。URLは以下のとおり。

		http://www.libsdl.org/release/SDL-devel-1.2.11-PPC.sea.bin

	   これを展開すると、 SDL-1.2.11-devel というような名前のフォルダが
	   できるので、このフォルダの名前を、 SDL-devel に変更してください。
		       ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
	   変更したら、このフォルダをまるごと  1) のフォルダ ( srcフォルダや、
	   QUASI88.make などのファイルのあるフォルダ ) にコピーします。


	3) QUASI88.make をダブルクリックすると、MPW が起動します。
	   メニューから「Build」→「Build」を選択してください。
	   ダイアログが表示されて、「Program Name?」と聞いてくるので
	   『QUASI88』と入力します。

	   コンパイルが始まり、実行ファイル QUASI88 が生成されます。

	   （ここで生成される実行ファイルには、
	     MAME/XMAMEベースのサウンド出力 と fmgen が組み込まれています。）


	   QUASI88 の実行には、 SDL のランタイムライブラリが必要です。

	   2) の開発者向けのライブラリの中 ( 例えば、 :SDL-1.2.11-devel:lib:
	   フォルダの中 ) に、SDL というファイルがあるので、これを実行ファイル
	   と同じフォルダに入れておきます。機能拡張フォルダにいれておいても
	   いいようです。

	   なお、このランタイムライブラリは、http://www.libsdl.org/ から単体
	   でも入手可能です。


	4) 出来上がった QUASI88 ですが、メモリが足りないらしく、メニューモード
	   に入ると固まったり、強制終了するという報告を受けています。
	   「情報を見る」(Command-I) にて、QUASI88 の メモリ必要条件を適当に
	   増やしてみてください。
	   デフォルトでは、5120K 程度になっていると思いますので、これを例えば
	   10000K ぐらいに増やしてみて下さい。


	5) ここからは、お好みで好きなように設定を変更してみてください。


	*) QUASI88 の改変・再配布などに関しては、もともとの QUASI88 の
	   著作権・免責事項に加えて以下を守ってください。

		QUASI88 の実行には、ランタイムライブラリ SDL が必要です。
		SDL をセットにして配布する場合は、 SDL に付属の
		README-SDL.txt をいっしょに配布してください。
		(SDLのドキュメントに、そう書いてあるので・・・)
		SDL のライセンスにも従ってください。

		MAME/XMAMEベースのサウンド出力を組み込んだ場合、
		MAME/XMAME のライセンスに従う必要があります。

		fmgenを組み込んだ場合、
		fmgen のライセンスに従う必要があります。

		実行ファイルと同じフォルダに、以下の空のフォルダを作成して
		おくとベターです。これらのフォルダは、設定のない状態で 
		QUASI88 を実行した時の各種参照フォルダになります。

			ROM   フォルダ
			DISK  フォルダ
			TAPE  フォルダ
			SNAP  フォルダ
			STATE フォルダ
