1. このプログラムについて

      このプログラムは、QUASI88 で使用されるディスクイメージファイルの
      ディスクイメージの以下の内容を表示します。

	・ヘッダの表示
	・セクタのIDの表示
	・セクタのデータの表示
	・ステータスが異常なセクタの表示
	・アンフォーマットトラックの表示
	・サイズが異常なセクタの表示

      注意) かなり適当に作ったので、バグが潜んでいるかもしれません。

      ・このソフトは無保証です。このソフトを使用した事によるいかなる損害も
        作者は保証いたしません。


2. コンパイル＆インストール

   インストーラを用意していませんので、

    % cc -o analyze88 analyze88.c

    でコンパイルした後、適当なディレクトリにインストールして下さい。


3. analyze88 の使用方法

    analyze88 [-header][-id][-data][-track n [m]][-image n]
		[-status] [-protect] filename

    	-header		… ヘッダ情報の詳細を表示します
	-id		… セクタのID情報を表示します。
	-data		… セクタのID情報とデータを表示します。
	-track n [m]	… チェックするトラックを指定します。
			   n が開始トラックで、省略時は 0 です。
			   m が終了トラックで、省略時は 163です。
	-image n	… チェックするイメージの番号を指定します。(1以上の値)
			   省略時は、全部のイメージが対象となります。
	-status		… セクタのステータスが異常なセクタを表示します。
			   (ステータス値が 0 以外のセクタを表示します。)
	-protect	… フォーマットが異常なトラックを表示します。
			   (現在は、アンフォーマットトラックおよび、セクタ
			    サイズが異常なセクタのみ。)
	filename	… ディスクイメージファイルを指定します。
			   省略時できません。

4. バグ

   トラック内のセクタ数のチェックが甘いです。
   ミックスレングスフォーマットの場合などは、正常に表示されないかも
   知れません。
